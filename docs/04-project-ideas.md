# 跟做项目清单

## Project 1：学习助教

输入一个主题，输出学习路径、推荐资料和练习题。

练习点：

- StateGraph
- 条件路由
- 简单 LLM 节点
- 输出结构化结果

## Project 2：资料检索助手

输入一个问题，从本地 Markdown 或网页资料中检索，再生成带来源的答案。

练习点：

- RAG
- retriever tool
- answer synthesis
- citation 约束

## Project 3：人工审批邮件助手

输入邮件上下文，agent 草拟回复，人类确认后才允许“发送”。

练习点：

- human-in-the-loop
- interrupt / resume
- 高风险动作审批

## Project 4：代码库问答助手

让 agent 读取一个小型代码库，回答“某功能在哪里实现”“调用链是什么”。

练习点：

- 工具调用
- 文件搜索
- 多轮状态
- 失败重试

## Project 5：Deep Research Mini

做一个简化版 deep research：planner 拆问题，researcher 查资料，writer 生成报告，critic 检查遗漏。

练习点：

- 多 agent
- subgraph
- research loop
- 评测样例
